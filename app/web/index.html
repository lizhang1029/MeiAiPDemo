分步评分向导：岗位 → 评分表 → 试题 → 音频 → 对应 → 评分

① 选择或新建岗位

岗位连同其评分表保存在本地（不依赖数据库），可重复使用。选择已有岗位会自动载入其评分表。

② 录入并解析评分表（文本）

粘贴整张评分表（评分项-满分-【权重】+ 子项 + 优秀/良好/合格/不合格 或 分数档位）。评分维度、满分、评分标准以此为准，总分=各项得分相加。

③ 录入并解析试题 JSON（仅取题干）

导入的 JSON 只提供题干。

④ 导入并解析音频（按题前停顿分段）

上传一名考生整场面试的一个音频/视频文件。系统转写后按「每题作答前的明显停顿」自动切分为多段，按题目顺序对应各题回答。

查看解析音频原文（按停顿分行）

⑤ 音频段 ↔ 试题 / 评分项 对应

「按题评分」项与试题、录音分段 1:1 对应（可调整）。「整场评分」项用全部转写、「人工录入」项由评委直接填分。

完成 ②（评分表）后，点「生成对应表」。

⑥ 开始 AI 评分

按评分表逐项打分：按题评分项剔除考官读题与「可以开始作答/时间到/下一题」等无关提示，仅对考生回答打分；分数为整数，给出剔除项与证据链。

评分结果与证据链

评分后在此显示总分（整数）、分项分、扣分原因、引用证据、AI 置信度，以及被剔除的考官读题/无关提示。
