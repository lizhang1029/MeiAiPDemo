整段面试 · 分步评分向导

① 上传整段面试录音并按题分段

上传一名考生整场面试的一个音频/视频文件。系统转写后，会按「每题作答前的明显停顿」自动切分为多段，按题目顺序对应各题回答。

查看整段原始转写

② 导入试题 JSON（仅取题干）

导入的 JSON 只提供题干；评分维度、满分与评分标准以下方手动录入为准。

③ 录入各维度评分标准与满分

每行 = 一题：题干只读、该题考生回答自动回填（可改）；维度名 / 满分 / 评分标准由你手动录入。总分以评分表合计为准。

完成 ① / ② 后，点「按题目数生成评分表」。

④ 开始评分

评分时自动剔除每段中混入的考官读题与「可以开始作答 / 时间到 / 下一题」等无关提示，仅对考生回答打分；所有分数为整数，并给出剔除项与证据链。

评分结果与证据链

评分后在此显示总分（整数）、分项分、扣分原因、引用证据、AI 置信度，以及被剔除的考官读题/无关提示。
